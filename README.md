# Tren-Gaji-Data-Analis
mendalami dinamika gaji seorang data analis, khususnya dalam konteks analisis sosial media di ranah politik kontemporer. Penelitian bertujuan untuk mengidentifikasi faktor-faktor yang memengaruhi struktur gaji para profesional ini, dengan fokus pada keterampilan khusus seperti Natural Language Processing (NLP).
